SECURITY CHECKLIST

실생활 보안 수칙

보안은 특별한 기술보다 매일 반복하는 작은 습관에서 시작됩니다.
아래 수칙들을 확인하고 안전한 인터넷 사용 습관을 만들어 봅시다.

ACCOUNT

계정 관리

여러 사이트에서 동일한 비밀번호를 사용하면 하나의 유출이 전체 계정 위험으로 이어집니다.

서비스마다 다른 비밀번호를 사용하고 반드시 2단계 인증을 설정해야 합니다.

특히 이메일 계정은 모든 계정의 중심이므로 가장 강하게 보호해야 합니다.

DEVICE

기기 보안

기기에는 개인정보와 로그인 정보가 저장되어 있기 때문에 매우 중요한 보안 대상입니다.

운영체제와 앱을 최신 상태로 유지하지 않으면 공격에 노출될 수 있습니다.

잠금 설정과 출처 불명의 프로그램 차단이 필수입니다.

PHISHING

피싱 대응

피싱 공격은 사용자를 속여 정보를 입력하게 만드는 방식입니다.

링크를 바로 클릭하지 말고 URL을 확인하거나 직접 접속해야 합니다.

긴급성을 강조하는 메시지는 대부분 위험 신호입니다.

DATA

데이터 보호

데이터는 한 번 유출되거나 손실되면 복구가 어려울 수 있습니다.

랜섬웨어에 대비해 중요한 파일은 반드시 백업해야 합니다.

민감한 정보는 안전하게 관리해야 합니다.

Action Guide

보안을 지키기 위해 해야 할 행동과 하지 말아야 할 행동

DO

해야 할 행동

👉 기본 원칙: 계정은 다르게, 인증은 강하게 관리한다.

🔐 계정 보안

  • ✅ 서비스마다 다른 비밀번호 사용하기
  • ✅ 2단계 인증(MFA) 설정하기
  • ✅ 로그인 알림 기능 활성화하기

💻 기기 보안

  • ✅ 운영체제와 앱 최신 상태 유지하기
  • ✅ 백신 프로그램 설치 및 주기적 검사
  • ✅ 잠금 화면 설정하기

🌐 인터넷 사용

  • ✅ 로그인 전 URL 주소 확인하기
  • ✅ 공용 PC 사용 후 반드시 로그아웃하기
  • ✅ 수상한 문자는 공식 사이트에서 확인하기

📁 데이터 보호

  • ✅ 중요한 파일 정기적으로 백업하기
  • ✅ 클라우드와 외장 저장장치 함께 활용하기
DON'T

하지 말아야 할 행동

⚠️ 기본 원칙: 급하게 누르거나, 아무 곳에 저장하지 않는다.

🚫 계정 관련 위험 행동

  • ❌ 같은 비밀번호를 여러 사이트에 사용하기
  • ❌ 공용 PC에 비밀번호 저장하기
  • ❌ 자동 로그인을 무조건 켜두기

⚠️ 피싱 위험 행동

  • ❌ 의심스러운 링크 바로 클릭하기
  • ❌ “무료”, “긴급” 메시지에 바로 반응하기
  • ❌ 발신자 확인 없이 첨부파일 열기

📡 네트워크 위험 행동

  • ❌ 공용 와이파이에서 금융 거래하기
  • ❌ 알 수 없는 와이파이에 자동 연결하기
  • ❌ 보안 경고를 무시하고 계속 진행하기

💾 데이터 위험 행동

  • ❌ 개인정보를 메모장에 저장하기
  • ❌ 출처 불명의 USB나 파일 실행하기
