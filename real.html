SECURITY CHECKLIST

실생활 보안 수칙

보안은 특별한 기술보다 매일 반복하는 작은 습관에서 시작됩니다.
아래 수칙들을 확인하고 안전한 인터넷 사용 습관을 만들어 봅시다.

전체
계정 관리

ACCOUNT

계정 관리

계정 탈취를 막는 첫 번째 방어선
  • 위험 비밀번호 재사용
  • 대응 서비스별 다른 비밀번호
  • 강화 2단계 인증(MFA)
  • 관리 로그인 기록 확인

하나의 계정이 해킹되면 이메일, SNS, 클라우드까지 연쇄적으로 침해될 수 있습니다. 특히 이메일 계정은 모든 계정의 중심이므로 가장 강하게 보호해야 합니다.

DEVICE

기기 보안

개인정보가 모여있는 핵심 장치
  • 위험 업데이트 미적용
  • 대응 OS 및 앱 최신 유지
  • 강화 잠금 화면 설정
  • 주의 출처 불명 앱 설치 금지

스마트폰과 PC는 인증 수단이기 때문에 침해되면 계정 보호가 무너질 수 있습니다. 특히 출처 불명의 앱이나 프로그램 설치는 악성코드 감염의 주요 원인이 됩니다.

PHISHING

피싱 대응

사용자의 심리를 노리는 공격
  • 위험 긴급/무료 문구
  • 대응 URL 주소 확인
  • 강화 공식 사이트 직접 접속
  • 확인 발신자 검증

피싱은 기술보다 심리를 이용하는 공격입니다. 링크를 바로 누르기보다 주소를 직접 입력하거나 공식 앱을 통해 접속하는 습관이 중요합니다.

DATA

데이터 보호

한 번 잃으면 되돌릴 수 없는 자산
  • 위험 랜섬웨어
  • 대응 정기 백업
  • 강화 클라우드 + 외장 저장
  • 관리 최소 정보 저장

데이터 유출은 장기간 악용될 수 있어 피해가 큽니다. 중요한 파일은 반드시 여러 위치에 분산 저장하고 민감한 정보는 최소한으로 보관해야 합니다.

Action Guide

보안을 지키기 위해 해야 할 행동과 하지 말아야 할 행동

DO

해야 할 행동

  • 서비스마다 다른 비밀번호 사용
  • 2단계 인증(MFA) 설정
  • OS 및 앱 최신 상태 유지
  • 백신 프로그램 정기 검사
  • URL 주소 확인 후 접속
  • 중요한 파일 정기 백업
계정은 다르게, 인증은 강하게, 파일은 반드시 백업한다.
DON'T

하지 말아야 할 행동

  • 의심스러운 링크 클릭
  • 비밀번호 저장 또는 공유
  • 출처 불명 파일 실행
  • 동일 비밀번호 반복 사용
  • 공용 와이파이에서 결제
  • 보안 경고 무시
급하게 누르지 말고, 아무 곳에 개인정보를 저장하지 않는다.
